# Fichiers du patch

Ce dossier contient les fichiers binaires qui remplacent ceux du jeu original :

| Fichier | Taille | Présent dans le repo ? | Présent dans la release ? |
|---|---|---|---|
| `script.mpk` | ~6,5 Mo | Oui (versionné dans Git) | Oui |
| `system.mpk` | ~133 Mo | **Non** (trop gros pour Git) | Oui |

**Pour installer le patch, téléchargez l'archive `.zip` complète depuis l'onglet
[Releases](../../releases/latest) du dépôt** — elle contient les deux fichiers prêts à
l'emploi, ainsi que l'installeur.

Cloner ce dépôt directement vous donnera tout sauf `system.mpk`. C'est suffisant pour
inspecter / corriger le code de l'installeur ou la documentation, mais pas pour faire une
installation complète.
